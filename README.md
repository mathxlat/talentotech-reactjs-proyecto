# Talento Tech - Entrega de Proyecto - Mathías Ezequiel Latrónico

## Acerca del proyecto

Aplicación web de e-commerce de hardware que muestra un catálogo de productos y permite agregarlos al carrito para simular una compra.

## Comandos

### Instalar dependencias

```bash
npm install
```

### Ejecutar

```bash
npm run dev
```

### Construir

```bash
npm run build
```

## Tecnologías

-   React + TypeScript (Vite)
-   React Router

## Funcionalidades

-   Ver catálogo de productos.
-   Ver detalle de producto en ruta dinámica.
-   Navegación por rutas con distintas secciones.
-   Agregar productos al carrito, eliminar ítems y vaciar carrito.
-   Manejo de estados de carga y error al obtener datos.
-   Manejo de efectos secundarios.
